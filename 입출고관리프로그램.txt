git status

*git push -u origin main 는 처음에만 1번.


git status  <-현재 변경사항 확인(수정된 파일이 뭔지 확인용)
git add -A <- 변경된 파일 전부 스테이징(수정/추가/삭제 전부 포함)
git commit -m "fix: server supabase env fallback" <-커밋
git push -u origin main <- GitHub로 push
